# YT Level — محلّل مستوى اللغة على YouTube

**حلّل مستوى CEFR (A1–C2) لأي فيديو على YouTube باستخدام الذكاء الاصطناعي المحلي — بدون مفاتيح API، بدون اتصال بالإنترنت.**

اختر بين محركي ذكاء اصطناعي: **Gemini Nano** (مدمج في Chrome) أو **Ollama** (خادم محلي). يعمل **لأي لغة** (الإنجليزية، الإسبانية، الفرنسية، الألمانية، الصينية، إلخ). تقوم الإضافة بجلب النص المكتوب (الترجمة) للفيديو وتصنيف مستواه حسب CEFR. تظهر شارة ملونة على كل صورة مصغّرة للفيديو.

## المميزات

- 🏷️ **شارات CEFR** — دوائر ملونة (A1–C2) على الصور المصغّرة لفيديوهات YouTube
- 🤖 **محركان للذكاء الاصطناعي** — استخدم **Gemini Nano** (AI المدمج في Chrome) أو **Ollama** (نماذج محلية)
- 🌍 **متعدد اللغات** — يحلل الفيديوهات بأي لغة
- 🎨 **خادم Ollama مخصص** — وجّه إلى أي مثيل Ollama على شبكتك
- ⚡ **تخزين مؤقت سريع** — يتم تخزين النتائج محلياً لتجنب إعادة التحليل
- 🔒 **خصوصية 100%** — كل شيء يعمل محلياً، لا تغادر أي بيانات جهازك

## المتطلبات

- **Chrome 128+**، **Brave**، أو أي متصفح يعتمد على Chromium
- **Gemini Nano**: Chrome 128+ مع تفعيل Prompt API (انظر أدناه)
- **Ollama**: Ollama مثبّت وقيد التشغيل ([ollama.com](https://ollama.com)) مع تنزيل نموذج واحد على الأقل

## التثبيت — Gemini Nano

Gemini Nano هو نموذج AI المدمج في Chrome. لا حاجة لتنزيلات أو خوادم.

### 1. تفعيل علامة Prompt API

1. افتح **`chrome://flags/#prompt-api-for-gemini-nano`**
2. اضبط العلامة على **"Enabled"**
3. انقر **"Relaunch"** لإعادة تشغيل Chrome

### 2. التحقق من حالة النموذج في الإضافة

1. انقر على أيقونة إضافة YT Level
2. اختر علامة التبويب **Gemini Nano**
3. ستظهر الحالة:
   - **Available** — جاهز للاستخدام
   - **Downloading** — يتم تنزيل النموذج
   - **Downloadable** — يحتاج إلى تنزيل أولاً (انقر لبدء التنزيل)
   - **Unavailable** — غير مدعوم في متصفحك

### 3. اختيار لغة التحليل

في علامة تبويب Gemini Nano، اختر لغة الفيديو الذي تريد تحليله:

| الرمز | اللغة       |
|-------|-------------|
| en    | الإنجليزية |
| es    | الإسبانية  |
| ja    | اليابانية  |
| de    | الألمانية  |
| fr    | الفرنسية   |

### 4. اختيار وضع الجهد

- **Quick** — تصنيف سريع باستخدام موجه بسيط
- **Deep** — تقييم CEFR مفصّل باستخدام موجه شامل

### 5. تحميل الإضافة

1. اذهب إلى **`chrome://extensions`** (أو **`brave://extensions`**)
2. فعّل **"وضع المطور"** (الزاوية العلوية اليمنى)
3. انقر **"تحميل الإضافة غير المضغوطة"**
4. حدد مجلد المشروع

### 6. منح الأذونات (هام)

1. في `chrome://extensions`، انقر **"التفاصيل"** على **YT Level**
2. فعّل **"السماح لهذه الإضافة بقراءة وتغيير جميع بياناتك على المواقع التي تزورها"**
3. إذا ظهرت رسالة، انقر **"السماح"**

> بدون هذه الخطوة، سيتم تحميل الإضافة لكنها لن تعمل على صفحات YouTube.

## التثبيت — Ollama

### 1. تثبيت Ollama

**Linux / macOS:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Windows:**
قم بتنزيل المثبت من [ollama.com/download](https://ollama.com/download) وقم بتشغيله. سيبدأ Ollama تلقائياً كخدمة خلفية.

### 2. تنزيل نموذج

```bash
ollama pull gemma3:1b
```

> يمكنك استخدام أي نموذج. تتيح لك الإضافة اختيار النموذج الذي تريد استخدامه من اللوحة المنبثقة.

### 3. إعداد CORS في Ollama

الإضافة تحتاج إلى إذن للتواصل مع Ollama من موقع YouTube الإلكتروني.

#### Linux — الخيار A: Systemd (دائم، موصى به)

```bash
sudo mkdir -p /etc/systemd/system/ollama.service.d

echo '[Service]
Environment=OLLAMA_ORIGINS=*' | sudo tee /etc/systemd/system/ollama.service.d/override.conf

sudo systemctl daemon-reload
sudo systemctl restart ollama
```

#### Linux — الخيار B: يدوي (مؤقت)

```bash
sudo systemctl stop ollama
OLLAMA_ORIGINS=* ollama serve
```

#### Windows — الخيار A: دائم (موصى به)

1. افتح **خصائص النظام** ← **متغيرات البيئة**
2. أضف متغير **نظام** جديد:
   - الاسم: `OLLAMA_ORIGINS`
   - القيمة: `*`
3. انقر **موافق** وأعد تشغيل Ollama من علبة النظام (انقر بزر الماوس الأيمن ← إنهاء، ثم ابدأ تشغيله مرة أخرى)

#### Windows — الخيار B: مؤقت (PowerShell)

```powershell
$env:OLLAMA_ORIGINS="*"
ollama serve
```

> في Windows، قم بتشغيل هذه الأوامر **بعد** إغلاق Ollama من علبة النظام.

### 4. تحميل الإضافة

نفس الخطوتين 5 و 6 في قسم Gemini Nano أعلاه.

### 5. استخدام الإضافة مع Ollama

1. انقر على أيقونة الإضافة
2. اختر علامة التبويب **Ollama**
3. حدد عنوان URL لخادمك (الافتراضي: `http://localhost:11434`)
4. انقر **موافق** لاختبار الاتصال
5. اختر نموذجاً من القائمة المنسدلة

## استخدام الإضافة

1. اذهب إلى **https://www.youtube.com**
2. الفيديوهات التي تحتوي على نصوص مكتوبة تظهر مؤشر تحميل أخضر أثناء التحليل
3. تظهر دائرة ملونة مع المستوى: **A1**، **A2**، **B1**، **B2**، **C1**، أو **C2**
4. مرر الماوس فوق الشارة لرؤية أي محرك ونموذج تم استخدامه
5. انقر على أيقونة الإضافة لفتح اللوحة المنبثقة والتبديل بين المحركات

## كيف يعمل

1. يستخرج معرف كل فيديو من قناة YouTube
2. يجلب النص المكتوب عبر `youtube-transcript.ai`
3. يرسل النص إلى محرك AI المحدد (Gemini Nano أو Ollama) لتصنيف CEFR
4. يعرض النتيجة كشارة دائرية على الصورة المصغّرة للفيديو
5. تُخزَّن النتائج محلياً لتجنب إعادة التحليل

## خادم Ollama مخصص

افتراضياً، تتصل الإضافة بـ `http://localhost:11434`. يمكنك تغيير ذلك:

1. انقر على أيقونة الإضافة
2. اختر علامة التبويب **Ollama**
3. أدخل عنوان URL لخادمك (مثال: `http://192.168.1.100:11434`)
4. انقر **موافق** — ستختبر الإضافة الاتصال وتحمّل النماذج المتاحة
5. انقر **↺** لإعادة التعيين إلى الإعداد الافتراضي

## ملاحظات

- يقوم فقط بتحليل الفيديوهات التي **تحتوي على نصوص مكتوبة متاحة** على YouTube
- وقت التحليل يعتمد على جهازك (أسرع مع Gemini Nano، 20–60 ثانية لكل فيديو على المعالج مع Ollama)
- إذا لم يكن أي محرك متاحاً، لن تظهر أي شارات
- لا حاجة لمفتاح API أو اتصال بالإنترنت
- جميع البيانات تبقى محلياً — لا شيء يُرسل إلى خوادم خارجية
