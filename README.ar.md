# YT Level — محلّل مستوى اللغة على YouTube

**حلّل مستوى CEFR (A1–C2) لأي فيديو على YouTube باستخدام الذكاء الاصطناعي المحلي — بدون مفاتيح API، بدون اتصال بالإنترنت.**

يعمل **لأي لغة** (الإنجليزية، الإسبانية، الفرنسية، الألمانية، الصينية، إلخ). تقوم الإضافة بجلب النص المكتوب (الترجمة) للفيديو وإرساله إلى نموذج Ollama المحلي لتصنيف CEFR. تظهر شارة ملونة على كل صورة مصغّرة للفيديو.

## المميزات

- 🏷️ **شارات CEFR** — دوائر ملونة (A1–C2) على الصور المصغّرة لفيديوهات YouTube
- 🤖 **ذكاء اصطناعي محلي** — يعمل مع أي نموذج Ollama (gemma، llama، mistral، إلخ)
- 🌍 **متعدد اللغات** — يحلل الفيديوهات بأي لغة
- 🎨 **خادم مخصص** — يمكنك توجيهه إلى أي مثيل Ollama على شبكتك
- ⚡ **تخزين مؤقت سريع** — يتم تخزين النتائج محلياً لتجنب إعادة التحليل
- 🔒 **خصوصية 100%** — كل شيء يعمل محلياً، لا تغادر أي بيانات جهازك

## المتطلبات

- **Chrome 128+**، **Brave**، أو أي متصفح يعتمد على Chromium
- **Ollama** مثبت وقيد التشغيل ([ollama.com](https://ollama.com))
- على الأقل **نموذج Ollama واحد** تم تنزيله (مثال: `ollama pull gemma3:1b`)

## التثبيت — خطوة بخطوة

### 1. تثبيت Ollama

**Linux / macOS:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Windows:**
قم بتنزيل المثبت من [ollama.com/download](https://ollama.com/download) وقم بتشغيله. سيبدأ Ollama تلقائياً كخدمة خلفية.

### 2. تنزيل نموذج

افتح الطرفية (موجه الأوامر في Windows) وقم بتشغيل:

```bash
ollama pull gemma3:1b
```

> يمكنك استخدام أي نموذج. تتيح لك الإضافة اختيار النموذج الذي تريد استخدامه من اللوحة المنبثقة.

### 3. إعداد CORS في Ollama

الإضافة تحتاج إلى إذن للتواصل مع Ollama من موقع YouTube الإلكتروني.

#### Linux — الخيار A: Systemd (دائم، موصى به)

```bash
sudo mkdir -p /etc/systemd/system/ollama.service.d

echo '[Service]
Environment=OLLAMA_ORIGINS=*' | sudo tee /etc/systemd/system/ollama.service.d/override.conf

sudo systemctl daemon-reload
sudo systemctl restart ollama
```

#### Linux — الخيار B: يدوي (مؤقت)

```bash
sudo systemctl stop ollama
OLLAMA_ORIGINS=* ollama serve
```

#### Windows — الخيار A: دائم (موصى به)

1. افتح خصائص النظام → متغيرات البيئة
2. أضف متغير نظام جديد:
   - الاسم: `OLLAMA_ORIGINS`
   - القيمة: `*`
3. انقر موافق وأعد تشغيل Ollama من علبة النظام (انقر بزر الماوس الأيمن → إنهاء، ثم ابدأ تشغيله مرة أخرى)

#### Windows — الخيار B: مؤقت (موجه الأوامر)

```cmd
set OLLAMA_ORIGINS=*
ollama serve
```

> في Windows، قم بتشغيل هذه الأوامر بعد إغلاق Ollama من علبة النظام.

### 4. تحميل الإضافة في المتصفح

1. اذهب إلى **`chrome://extensions`** (أو **`brave://extensions`**)
2. فعّل **"وضع المطور"** (الزاوية العلوية اليمنى)
3. انقر **"تحميل الإضافة غير المضغوطة"**
4. حدد مجلد المشروع

### 5. منح الأذونات (هام)

بعض المتصفحات تتطلب إذناً صريحاً للإضافة لكي تعمل على المواقع:

1. في `chrome://extensions`، انقر **"التفاصيل"** على **YT Level**
2. فعّل **"السماح لهذه الإضافة بقراءة وتغيير جميع بياناتك على المواقع التي تزورها"**
3. إذا ظهرت رسالة، انقر **"السماح"**

> بدون هذه الخطوة، سيتم تحميل الإضافة لكنها لن تعمل على صفحات YouTube.

### 6. استخدام الإضافة

1. اذهب إلى **https://www.youtube.com**
2. الفيديوهات التي تحتوي على نصوص مكتوبة تظهر مؤشر تحميل أخضر أثناء التحليل
3. تظهر دائرة ملونة مع المستوى: **A1**، **A2**، **B1**، **B2**، **C1**، أو **C2**
4. مرر الماوس فوق الشارة لرؤية أي نموذج تم استخدامه
5. انقر على أيقونة الإضافة لفتح اللوحة المنبثقة:
   - **الخادم** — تغيير عنوان URL لخادم Ollama الخاص بك إذا لزم الأمر
   - **النموذج** — اختيار النموذج المثبت الذي تريد استخدامه
   - **اللغة** — تغيير لغة واجهة الإضافة

## كيف يعمل

1. يستخرج معرف كل فيديو من قناة YouTube
2. يجلب النص المكتوب عبر `youtube-transcript.ai`
3. يرسل النص إلى نموذج Ollama المحلي الخاص بك لطلب تصنيف CEFR
4. يعرض النتيجة كشارة دائرية على الصورة المصغّرة للفيديو
5. تُخزَّن النتائج محلياً لتجنب إعادة التحليل

## خادم Ollama مخصص

افتراضياً، تتصل الإضافة بـ `http://localhost:11434`. يمكنك تغيير ذلك:

1. انقر على أيقونة الإضافة
2. أدخل عنوان URL لخادمك (مثال: `http://192.168.1.100:11434`)
3. انقر **موافق** — ستختبر الإضافة الاتصال وتحمّل النماذج المتاحة
4. انقر **↺** لإعادة التعيين إلى الإعداد الافتراضي

## ملاحظات

- يقوم فقط بتحليل الفيديوهات التي **تحتوي على نصوص مكتوبة متاحة** على YouTube
- وقت التحليل يعتمد على جهازك وحجم النموذج (20–60 ثانية لكل فيديو على المعالج)
- إذا كان Ollama لا يعمل أو لم يتم تثبيت أي نموذج، لن تظهر أي شارات
- لا حاجة لمفتاح API أو اتصال بالإنترنت (بعد تنزيل النموذج)
- جميع البيانات تبقى محلياً — لا شيء يُرسل إلى خوادم خارجية
